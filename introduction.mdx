# Welcome to Shadcn Admin

Shadcn Admin is a modern, responsive admin dashboard UI crafted with **Shadcn UI** and **Vite**. Built with accessibility, responsiveness, and customization in mind, it provides a comprehensive collection of reusable components and page templates for building admin interfaces.

## Tech Stack

Shadcn Admin is built with modern, production-ready technologies:

- **UI Framework**: [Shadcn UI](https://ui.shadcn.com) (TailwindCSS + Radix UI)
- **Build Tool**: [Vite](https://vitejs.dev/) for lightning-fast builds
- **Routing**: [TanStack Router](https://tanstack.com/router/latest) with type-safe navigation
- **Type Checking**: [TypeScript](https://www.typescriptlang.org/) for enhanced developer experience
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/) for simple, scalable state
- **Data Fetching**: [TanStack Query](https://tanstack.com/query/latest) for server state
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) with [Zod](https://zod.dev/) validation
- **Charts**: [Recharts](https://recharts.org/) for beautiful data visualizations
- **Icons**: [Lucide Icons](https://lucide.dev/) for consistent iconography
