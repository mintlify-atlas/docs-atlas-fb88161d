The Chats page provides a messaging interface with a split-pane layout showing a chat list and conversation view, including a searchable new chat dialog.

## Overview

The Chats component implements a responsive messaging UI with real-time conversation display, date grouping, and user search functionality.

**Location:** `src/features/chats/index.tsx`

## Date Grouping Messages

Messages are grouped by date using a reducer pattern:

```tsx
const currentMessage = selectedUser?.messages.reduce(
  (acc: Record<string, Convo[]>, obj) => {
    const key = format(obj.timestamp, 'd MMM, yyyy')

    // Create an array for the category if it doesn't exist
    if (!acc[key]) {
      acc[key] = []
    }

    // Push the current object to the array
    acc[key].push(obj)

    return acc
  },
  {}
)
```

This creates an object where keys are date strings and values are arrays of messages:

```js
{
  "14 Mar, 2026": [
    { sender: "You", message: "Hello!", timestamp: ... },
    { sender: "John", message: "Hi there!", timestamp: ... }
  ],
  "13 Mar, 2026": [
    { sender: "You", message: "See you tomorrow", timestamp: ... }
  ]
}
```

## New Chat Dialog

The NewChat component provides a searchable user selection dialog:

```tsx src/features/chats/components/new-chat.tsx
export function NewChat({ users, onOpenChange, open }: NewChatProps) {
  const [selectedUsers, setSelectedUsers] = useState<User[]>([])

  const handleSelectUser = (user: User) => {
    if (!selectedUsers.find((u) => u.id === user.id)) {
      setSelectedUsers([...selectedUsers, user])
    } else {
      handleRemoveUser(user.id)
    }
  }

  return (
    <Dialog open={open} onOpenChange={handleOpenChange}>
      <DialogContent className='sm:max-w-[600px]'>
        <DialogHeader>
          <DialogTitle>New message</DialogTitle>
        </DialogHeader>
        <div className='flex flex-col gap-4'>
          {/* Selected Users */}
          <div className='flex flex-wrap gap-2'>
            <span className='text-sm text-muted-foreground'>To:</span>
            {selectedUsers.map((user) => (
              <Badge key={user.id} variant='default'>
                {user.fullName}
                <button onClick={() => handleRemoveUser(user.id)}>
                  <X className='h-3 w-3' />
                </button>
              </Badge>
            ))}
          </div>

          {/* User Search */}
          <Command className='rounded-lg border'>
            <CommandInput placeholder='Search people...' />
            <CommandList>
              <CommandEmpty>No people found.</CommandEmpty>
              <CommandGroup>
                {users.map((user) => (
                  <CommandItem
                    key={user.id}
                    onSelect={() => handleSelectUser(user)}
                  >
                    <div className='flex items-center gap-2'>
                      <img
                        src={user.profile}
                        alt={user.fullName}
                        className='h-8 w-8 rounded-full'
                      />
                      <div>
                        <span className='text-sm font-medium'>
                          {user.fullName}
                        </span>
                        <span className='text-xs'>{user.username}</span>
                      </div>
                    </div>
                    {selectedUsers.find((u) => u.id === user.id) && (
                      <Check className='h-4 w-4' />
                    )}
                  </CommandItem>
                ))}
              </CommandGroup>
            </CommandList>
          </Command>

          <Button
            onClick={() => showSubmittedData(selectedUsers)}
            disabled={selectedUsers.length === 0}
          >
            Chat
          </Button>
        </div>
      </DialogContent>
    </Dialog>
  )
}
```

## Customization

### Changing Message Bubble Styles

Modify the message appearance:

```tsx
<div
  className={cn(
    'max-w-80 px-4 py-3 shadow-md', // Changed max-width and padding
    msg.sender === 'You'
      ? 'self-end rounded-2xl bg-blue-500 text-white' // Custom colors
      : 'self-start rounded-2xl bg-gray-100 text-gray-900'
  )}
>
```

### Adding Message Actions

Implement message-level actions:

```tsx
<div className='group relative'>
  <div className='message-bubble'>{msg.message}</div>
  <div className='absolute right-0 top-0 hidden group-hover:flex'>
    <Button size='sm' variant='ghost'>
      <Reply size={14} />
    </Button>
    <Button size='sm' variant='ghost'>
      <Copy size={14} />
    </Button>
  </div>
</div>
```

