The `ThemeSwitch` component provides a user-friendly dropdown menu for selecting the application theme. It automatically updates the theme-color meta tag and displays visual indicators for the active theme.

## Implementation

```tsx src/components/theme-switch.tsx
import { useEffect } from 'react'
import { Check, Moon, Sun } from 'lucide-react'
import { cn } from '@/lib/utils'
import { useTheme } from '@/context/theme-provider'
import { Button } from '@/components/ui/button'
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from '@/components/ui/dropdown-menu'

export function ThemeSwitch() {
  const { theme, setTheme } = useTheme()

  /* Update theme-color meta tag
   * when theme is updated */
  useEffect(() => {
    const themeColor = theme === 'dark' ? '#020817' : '#fff'
    const metaThemeColor = document.querySelector("meta[name='theme-color']")
    if (metaThemeColor) metaThemeColor.setAttribute('content', themeColor)
  }, [theme])

  return (
    <DropdownMenu modal={false}>
      <DropdownMenuTrigger asChild>
        <Button variant='ghost' size='icon' className='scale-95 rounded-full'>
          <Sun className='size-[1.2rem] scale-100 rotate-0 transition-all dark:scale-0 dark:-rotate-90' />
          <Moon className='absolute size-[1.2rem] scale-0 rotate-90 transition-all dark:scale-100 dark:rotate-0' />
          <span className='sr-only'>Toggle theme</span>
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align='end'>
        <DropdownMenuItem onClick={() => setTheme('light')}>
          Light{' '}
          <Check
            size={14}
            className={cn('ms-auto', theme !== 'light' && 'hidden')}
          />
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme('dark')}>
          Dark
          <Check
            size={14}
            className={cn('ms-auto', theme !== 'dark' && 'hidden')}
          />
        </DropdownMenuItem>
        <DropdownMenuItem onClick={() => setTheme('system')}>
          System
          <Check
            size={14}
            className={cn('ms-auto', theme !== 'system' && 'hidden')}
          />
        </DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  )
}
```

## Theme Options

### Light Theme

```tsx
<DropdownMenuItem onClick={() => setTheme('light')}>
  Light
</DropdownMenuItem>
```

Sets the application to light mode with white backgrounds and dark text.

### Dark Theme

```tsx
<DropdownMenuItem onClick={() => setTheme('dark')}>
  Dark
</DropdownMenuItem>
```

Sets the application to dark mode with dark backgrounds and light text.

### System Theme

```tsx
<DropdownMenuItem onClick={() => setTheme('system')}>
  System
</DropdownMenuItem>
```

Follows the user's operating system theme preference. Automatically switches between light and dark based on OS settings.

## Meta Tag Updates

The component automatically updates the `theme-color` meta tag when the theme changes:

```tsx
useEffect(() => {
  const themeColor = theme === 'dark' ? '#020817' : '#fff'
  const metaThemeColor = document.querySelector("meta[name='theme-color']")
  if (metaThemeColor) metaThemeColor.setAttribute('content', themeColor)
}, [theme])
```

**Color values:**
- **Dark theme**: `#020817` (slate-950)
- **Light theme**: `#fff` (white)

This updates the browser chrome color on mobile devices.

## Icon Animations

The trigger button features animated sun/moon icons:

```tsx
<Sun className='size-[1.2rem] scale-100 rotate-0 transition-all dark:scale-0 dark:-rotate-90' />
<Moon className='absolute size-[1.2rem] scale-0 rotate-90 transition-all dark:scale-100 dark:rotate-0' />
```

**Animation behavior:**
- **Light mode**: Sun icon visible (scale 100%, rotate 0°), Moon hidden (scale 0%, rotate 90°)
- **Dark mode**: Sun hidden (scale 0%, rotate -90°), Moon visible (scale 100%, rotate 0°)
- **Transition**: Smooth scale and rotation transitions

## Active Indicator

Each menu item shows a checkmark when active:

```tsx
<Check
  size={14}
  className={cn('ms-auto', theme !== 'light' && 'hidden')}
/>
```

The check icon:
- Positioned at `ms-auto` (margin-inline-start: auto, right-aligned in LTR)
- Size 14px
- Hidden when not the active theme

### Programmatic Theme Change

Change theme from anywhere in your app:

```tsx
import { useTheme } from '@/context/theme-provider'

function AutoThemeToggle() {
  const { theme, setTheme } = useTheme()
  
  useEffect(() => {
    const hour = new Date().getHours()
    // Auto dark mode from 8 PM to 6 AM
    if (hour >= 20 || hour < 6) {
      setTheme('dark')
    } else {
      setTheme('light')
    }
  }, [])
  
  return null
}
```

## Comparison with Config Drawer

| Feature | ThemeSwitch | ConfigDrawer |
|---------|-------------|-------------|
| **Purpose** | Quick theme toggle | Comprehensive settings |
| **Location** | Header/navbar | Settings panel |
| **Settings** | Theme only | Theme, sidebar, layout, direction |
| **UI** | Dropdown menu | Sheet drawer |
| **Persistence** | ThemeProvider | Multiple providers |

**Use ThemeSwitch when:**
- You want a compact theme toggle in the header
- Theme is the primary customization option
- Space is limited

**Use ConfigDrawer when:**
- You have multiple customization options
- You want to group all settings together
- You have space for a settings button
