## Configuration

All components are configured via `components.json`:

```json components.json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "new-york",
  "rsc": false,
  "tsx": true,
  "tailwind": {
    "config": "",
    "css": "src/styles/index.css",
    "baseColor": "slate",
    "cssVariables": true,
    "prefix": ""
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  },
  "iconLibrary": "lucide"
}
```

### Key Settings

<ParamField path="style" type="string" default="new-york">

<ParamField path="rsc" type="boolean" default="false">

### Sidebar RTL Support

The sidebar component includes RTL-specific styles:

```tsx src/components/ui/sidebar.tsx
<SidebarRail
  className={cn(
    // RTL support
    'rtl:translate-x-1/2',
    'rtl:in-data-[side=left]:cursor-e-resize rtl:in-data-[side=right]:cursor-w-resize',
    'rtl:[[data-side=left][data-state=collapsed]_&]:cursor-w-resize'
  )}
/>
```

**RTL Features:**
- Mirrored rail positioning
- Reversed cursor resize directions
- Flipped chevron rotations

### Logical Properties

All spacing and positioning uses CSS logical properties:

```tsx
// Physical properties (old)
margin-left: 1rem
padding-right: 2rem

// Logical properties (RTL-compatible)
margin-inline-start: 1rem  // ms-4
padding-inline-end: 2rem   // pe-8
```

**Tailwind logical classes:**
- `ms-*` - margin-inline-start (left in LTR, right in RTL)
- `me-*` - margin-inline-end (right in LTR, left in RTL)
- `ps-*` - padding-inline-start
- `pe-*` - padding-inline-end
- `start-*` - inset-inline-start
- `end-*` - inset-inline-end

```css src/styles/index.css

  --foreground: 222.2 84% 4.9%;
  --primary: 222.2 47.4% 11.2%;
  --primary-foreground: 210 40% 98%;
```

## Adding New Components

Use the Shadcn CLI to add components:

```bash
npx shadcn@latest add [component-name]
```

**Example:**
```bash
# Add the accordion component
npx shadcn@latest add accordion

# Add multiple components
npx shadcn@latest add hover-card menubar
```

Components are automatically:
- Added to `src/components/ui/`
- Configured with project settings
- Styled with the New York variant
- Compatible with the existing theme

## Customization

### Modify Components

All components live in `src/components/ui/` and can be modified:

```tsx src/components/ui/button.tsx
// Add a new variant
const buttonVariants = cva(
  'base-styles',
  {
    variants: {
      variant: {
        default: '...',
        custom: 'bg-purple-500 text-white', // Your custom variant
      },
    },
  }
)
```

### Extend with New Props

```tsx
import { Button, type ButtonProps } from '@/components/ui/button'

interface CustomButtonProps extends ButtonProps {
  loading?: boolean
}

export function CustomButton({ loading, children, ...props }: CustomButtonProps) {
  return (
    <Button disabled={loading} {...props}>
      {loading ? 'Loading...' : children}
    </Button>
  )
}
```
