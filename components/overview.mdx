## Layout Components

Layout components form the structural foundation of your admin dashboard:

- **[App Sidebar](/components/app-sidebar)** - Collapsible navigation sidebar with team switcher and navigation groups
- **[Header](/components/header)** - Sticky header with blur effect on scroll
- **[Main Layout](/components/main-layout)** - Main content container with fixed and fluid layout options

## UI Components

Shadcn Admin includes all 30+ Shadcn UI components with:

- **RTL Support** - Fully tested right-to-left layout support
- **New York Style** - Consistent design system
- **Customized Components** - Enhanced for admin use cases
- **Type Safety** - Full TypeScript support

View the complete [UI Components](/components/ui-components) documentation.

## Integration with Shadcn UI

All components are built using the Shadcn UI CLI configuration:

```json components.json
{
  "style": "new-york",
  "rsc": false,
  "tsx": true,
  "tailwind": {
    "baseColor": "slate",
    "cssVariables": true
  },
  "iconLibrary": "lucide"
}
```

## Importing Components

Components use path aliases for clean imports:

```tsx
import { AppSidebar } from '@/components/layout/app-sidebar'
import { CommandMenu } from '@/components/command-menu'
import { Button } from '@/components/ui/button'
```
