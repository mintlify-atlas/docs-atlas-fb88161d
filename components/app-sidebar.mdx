The `AppSidebar` component provides a responsive, collapsible sidebar for application navigation. It supports multiple variants, states, and includes team switching, grouped navigation, and user profile sections.

## Usage

```tsx src/components/layout/app-sidebar.tsx
import { AppSidebar } from '@/components/layout/app-sidebar'

export function Layout() {
  return (
    <SidebarProvider>
      <AppSidebar />
      <SidebarInset>
        {/* Your content */}
      </SidebarInset>
    </SidebarProvider>
  )
}
```

## Structure

The sidebar is composed of three main sections:

```tsx
export function AppSidebar() {
  const { collapsible, variant } = useLayout()
  return (
    <Sidebar collapsible={collapsible} variant={variant}>
      <SidebarHeader>
        <TeamSwitcher teams={sidebarData.teams} />
      </SidebarHeader>
      <SidebarContent>
        {sidebarData.navGroups.map((props) => (
          <NavGroup key={props.title} {...props} />
        ))}
      </SidebarContent>
      <SidebarFooter>
        <NavUser user={sidebarData.user} />
      </SidebarFooter>
      <SidebarRail />
    </Sidebar>
  )
}
```

### Navigation Items

Navigation items can be either links or collapsible groups:

```typescript src/components/layout/types.ts
type NavLink = BaseNavItem & {
  url: LinkProps['to'] | (string & {})
  items?: never
}

type NavCollapsible = BaseNavItem & {
  items: (BaseNavItem & { url: LinkProps['to'] | (string & {}) })[]
  url?: never
}

type NavItem = NavCollapsible | NavLink
```

## Layout Options

Control sidebar behavior using the `LayoutProvider`:

### Collapsible Modes

<ParamField path="collapsible" type="'offcanvas' | 'icon' | 'none'" default="icon">
  Controls how the sidebar collapses:
  
  - `offcanvas` - Sidebar slides off-screen when collapsed
  - `icon` - Sidebar shows only icons when collapsed  
  - `none` - Sidebar cannot be collapsed
</ParamField>

### Variants

<ParamField path="variant" type="'inset' | 'sidebar' | 'floating'" default="inset">
  Visual style of the sidebar:
  
  - `inset` - Sidebar with rounded content area
  - `sidebar` - Standard edge-to-edge sidebar
  - `floating` - Floating sidebar with shadow
</ParamField>

```tsx
import { useLayout } from '@/context/layout-provider'

function MyComponent() {
  const { collapsible, variant, setCollapsible, setVariant } = useLayout()
  
  return (
    <div>
      <button onClick={() => setCollapsible('icon')}>
        Icon Mode
      </button>
      <button onClick={() => setVariant('floating')}>
        Floating Style
      </button>
    </div>
  )
}
```

## Keyboard Shortcuts

The sidebar includes built-in keyboard shortcuts:

- **Cmd/Ctrl + B** - Toggle sidebar open/closed

## Responsive Behavior

- **Desktop (md+)**: Sidebar renders with selected variant and collapsible mode
- **Mobile**: Sidebar renders as a Sheet overlay, toggled by SidebarTrigger

## Customization

### Using App Title Instead of Team Switcher

Replace the TeamSwitcher with a simple app title:

```tsx src/components/layout/app-sidebar.tsx
import { AppTitle } from './app-title'

<SidebarHeader>
  <AppTitle />
  {/* Instead of: <TeamSwitcher teams={sidebarData.teams} /> */}
</SidebarHeader>
```

### Custom Navigation Groups

Create custom navigation by modifying `sidebarData`:

```tsx src/components/layout/data/sidebar-data.ts
import { Home, Settings, Users } from 'lucide-react'

export const sidebarData = {
  navGroups: [
    {
      title: 'Main',
      items: [
        {
          title: 'Dashboard',
          url: '/',
          icon: Home,
        },
        {
          title: 'Team',
          icon: Users,
          items: [
            { title: 'Members', url: '/team/members' },
            { title: 'Roles', url: '/team/roles' },
          ],
        },
      ],
    },
  ],
}
```
